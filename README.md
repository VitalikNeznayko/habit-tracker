# Habit Tracker

Habit Tracker is a fullstack web application for tracking daily habits and monitoring user progress.

## Features

- Google OAuth authentication
- JWT authorization
- Create, edit and delete habits
- Daily habit check-ins
- Streak tracking
- Progress statistics
- Heatmap habit
- Responsive UI

## Technologies

- Next.js
- TypeScript
- Prisma
- PostgreSQL
- Tailwind CSS
- JWT
- Google OAuth

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```env
DATABASE_URL=

JWT_SECRET=
JWT_REFRESH_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

NEXT_PUBLIC_APP_URL=
```

## Run Development Server

```bash
npm run dev
```

## Build Project

```bash
npm run build
```

## Deployment

The project is deployed on Vercel.
